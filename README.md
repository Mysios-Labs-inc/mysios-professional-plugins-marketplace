## 🔗 Integrated Professional Intelligence Workflows

**Career + Market Intelligence:**
```bash
# Research target companies
strategic-research-intelligence:competitive-analysis competitors="target-company.com"

# Find opportunities at researched companies
career-intelligence:job-search company="Target Company" industry="researched-sector"

# Create targeted applications based on research insights
career-intelligence:resume-optimize role="specific-role" company-insights="research-findings"
```

**Market Trend → Career Positioning:**
```bash
# Monitor industry trends
strategic-research-intelligence:trend-monitor topic="AI industry trends" timeframe="month"

# Identify emerging job categories
career-intelligence:market-analysis trends="AI-trends" skills-gap-analysis=true

# Position for future opportunities
career-intelligence:skill-development-plan based-on="market-intelligence"
```
